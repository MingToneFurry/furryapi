The main site image API supports the following optional query parameters:
1. format=json|file The former returns the image link in JSON format, the latter directly returns the image file
2. redirect=1|0 The former issues a 302 redirect to the image URL, the latter does not perform a redirect
3. mode=auto|box|large Changes the returned image mode: auto = automatic detection, box = portrait image, large = landscape image
